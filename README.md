# CASHEX — Telegram-бот обменника

Бот для обменника криптовалюты CASHEX с кнопками: **О нас**, **Курсы**, **Связаться с нами**, **Как нас найти**.

## Установка

1. Создайте бота в Telegram через [@BotFather](https://t.me/BotFather), получите токен.
2. Клонируйте/скопируйте проект и перейдите в папку:
   ```bash
   cd Cashex
   ```
3. Создайте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
5. Скопируйте пример конфига и заполните данные:
   ```bash
   copy .env.example .env
   ```
   В файле `.env` укажите:
   ```
   BOT_TOKEN=ваш_токен_от_BotFather
   ```
6. Отредактируйте `config.py`: укажите адрес, телефон, email, Telegram и при необходимости курсы.

## Запуск

```bash
python bot.py
```

После запуска напишите боту в Telegram команду `/start`.

## Настройка контента

В файле **config.py** задайте:

- `ADDRESS` — адрес офиса (для приветствия и «Как нас найти»).
- `CONTACT_PHONE`, `CONTACT_EMAIL`, `CONTACT_TELEGRAM` — контакты для «Связаться с нами».
- `ABOUT_YEARS` — сколько лет на рынке (для блока «О нас»).
- `RATE_BUY_USDT`, `RATE_SELL_USDT` — курсы покупки/продажи USDT (или «по запросу»).

## Команды и кнопки

| Действие | Описание |
|----------|----------|
| `/start` | Приветствие и главное меню с кнопками |
| `/exchange` | Заглушка «Обмен» (можно развить в создание заявки) |
| **О нас** | Информация о компании и адрес |
| **Курсы** | Текущие курсы обмена |
| **Связаться с нами** | Телефон, email, Telegram |
| **Как нас найти** | Адрес и режим работы |

Дальше можно добавить логику создания заявок на обмен и, при необходимости, получение курсов из API.
